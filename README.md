# AI Mock Interview Simulator

https://singhnandini60810-sys.github.io/Mock-Interview-Simulator/

A browser-based AI-powered interview practice application built using HTML, CSS, JavaScript, and the Gemini API.

## Features

* AI-generated interview questions
* Multiple job roles
* AI answer evaluation
* Feedback and scoring
* Follow-up questions
* Responsive UI
* Deployable on GitHub Pages

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Gemini API

## Setup

1. Clone the repository.
2. Open index.html.
3. Generate a Gemini API key from Google AI Studio.
4. Paste the API key into the application.
5. Start the interview.
