# Mock Interview Simulator

A browser-based interview practice platform built using HTML, CSS, and JavaScript. The application allows users to simulate technical interviews by selecting a job role and answering a series of questions in an interactive environment.

## Features

- Select interview role
  - Frontend Developer
  - Python Developer
- Multiple interview questions
- Interactive question-by-question flow
- Answer submission system
- Progress tracking
- Automatic interview completion summary
- Basic scoring based on answer completion
- Fully client-side application
- Deployable on GitHub Pages

## Technologies Used

- HTML5
- CSS3
- JavaScript
